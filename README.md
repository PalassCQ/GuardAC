# GuardAC

**AI-анти-чит для Minecraft.**
Ловит аим-хаки нейросетью по физике движения прицела.

</div>

---

GuardAC — плагин для Spigot / Paper / Folia, который отправляет короткие окна
поведения прицела на инференс-сервис GuardAC и по ответу нейросети (вероятность
чита) поднимает алерты стаффу и/или выдаёт наказания. Модель распознаёт аимботы,
аим-ассисты и «closet»-читеров, которых обычные проверки пропускают.

Этот репозиторий — **сам плагин** (открытый, GPLv3). Сервис инференса (модель,
ключи, тарифы) — отдельный SaaS; ключ для подключения берётся на сайте GuardAC.

## Возможности

- Нейросетевая проверка прицела в реальном времени (ответ за миллисекунды).
- Безопасный режим `only-alert` — сначала только оповещения, без банов.
- Буфер подозрений с VL, порогами и затуханием — устойчив к разовому шуму.
- «Отпечаток прицела»: ловит игрока, включившего чит посреди сессии.
- Алерты стаффу (кликабельные), голограммы, меню подозреваемых, наказания
  с анимациями — всё настраивается.
- Обход Bedrock (Geyser) и регионов WorldGuard, лаг-компенсация по телепортам,
  анти-релог, кросс-серверная репутация.

## Установка

1. Собери jar:
   ```bash
   ./gradlew build      # результат в build/libs/GuardAC-*.jar
   ```
2. Положи jar в папку `plugins/` сервера и перезапусти его.
3. В `plugins/GuardAC/config.yml` укажи адрес сервиса и свой ключ:
   ```yaml
   ai:
     server: "https://guardac.net"
     api-key: "ВСТАВЬ-СВОЙ-КЛЮЧ"
   ```
   Ключ можно получить на сайте GuardAC.

## Сборка

- JDK 17+ и системный Gradle (или входящий `./gradlew`).
- `./gradlew build` — компиляция, тесты и сборка jar.

## Совместимость

Minecraft 1.21.x (Spigot / Paper / Folia), Java 17+. Bedrock-игроки через
Geyser автоматически исключаются из проверки.
